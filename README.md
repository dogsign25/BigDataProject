# 🏠 서울 부동산 임대차 데이터 분석 대시보드 (Streamlit) 📌

## 프로젝트 개요
이 프로젝트는 서울시 전월세 계약 데이터(`seoul.csv`)를 기반으로 구축된 대화형 **Streamlit** 대시보드입니다. 이 대시보드는 사용자에게 핵심 시장 지표, 가격 효율성 분석, 시장 트렌드, 그리고 잠재적 리스크에 대한 심층적인 분석 결과를 제공하여 데이터 기반의 합리적인 부동산 의사결정을 지원하는 것을 목표로 합니다.

## 주제 및 주제 선택 이유
### 주제
서울시 전월세 시장의 가격 및 트렌드를 분석하여 부동산 투자자와 임차인, 정책 입안자들이 실시간으로 유용한 시장 정보를 얻을 수 있도록 하는 대시보드입니다.

### 주제 선택 이유
서울은 국내에서 가장 중요한 부동산 시장을 형성하고 있으며, 최근 몇 년간 전월세 시장의 변화가 큰 이슈로 떠오르고 있습니다. 이 분석은 부동산 관련 의사결정을 지원하고, 시장 내 가격 변화 및 리스크를 예측할 수 있는 도구를 제공합니다. 실시간으로 변동하는 부동산 시장을 데이터 기반으로 분석할 수 있는 대시보드가 필요하다고 판단하여 이 주제를 선택하게 되었습니다.

## 데이터 분석 내용
이 프로젝트에서는 `seoul.csv` 데이터를 기반으로 네 가지 핵심 분석 기능을 제공합니다:

### 1. **가격 효율성 및 분포 분석**
- **목표:** 1㎡당 가격 (평당 가격)을 계산하여 자치구별 효율성을 비교하고, 비효율적인 거래를 탐색합니다.
- **분석 내용:** 가격 분포, 구별 효율성 지표를 분석하여 사용자가 효율적인 투자 지역을 파악할 수 있도록 돕습니다.

### 2. **시장 트렌드 분석**
- **목표:** 시간이 지남에 따라 변동되는 서울시의 임대차 시장 트렌드를 분석합니다.
- **분석 내용:** 선택된 지역/연도의 월별 평균 가격 및 계약 건수 변화 추이를 시각화하여 시장 변화를 직관적으로 확인할 수 있습니다.

### 3. **심화 맞춤 비교 분석**
- **목표:** 사용자가 정의한 두 시장 그룹 간의 가격, 면적, 회귀선 기울기 비교를 제공합니다.
- **분석 내용:** 예를 들어, "강남구 아파트 전세" vs "강북구 연립 월세"와 같이 비교 분석을 통해 가격 및 시장 트렌드를 비교할 수 있습니다.

### 4. **리스크 및 노후도 분석**
- **목표:** 가격 리스크 및 건물 노후도에 따른 리스크를 분석합니다.
- **분석 내용:** 통계적 이상치(Outlier)를 탐색하여 가격 리스크를 경고하고, 건물의 노후도에 따른 추가 리스크를 평가합니다.

## 사용 방법 (How to Run)

### 1. 환경 설정

#### 필요 라이브러리 설치:
```bash
pip install streamlit pandas plotly numpy
```
#### 데이터 파일 준비:
- 분석에 사용할 seoul.csv 파일을 프로젝트의 루트 디렉토리에 위치시킵니다. (파일명이 다를 경우, app.py의 FILE_PATH 변수를 수정해야 합니다.)


# 프로젝트 파일 구조:
bash
```
real_estate_dashboard/
├── app.py                # 홈 화면 및 데이터 로드 정의
├── seoul.csv             # 원본 데이터 파일
└── pages/
    ├── 1_Analysis_Dashboard.py  # 시장 현황 및 KPI 요약
    ├── 3_Comparative_Analysis.py  # 심화 맞춤 비교 분석
    └── 4_Risk_and_Forecast.py    # 리스크 및 노후도 분석
```

### 2. 대시보드 실행
- 프로젝트 루트 디렉토리로 이동한 후, 터미널에서 다음 명령어를 실행합니다:
```
streamlit run app.py
```
- 명령어를 실행하면, 웹 브라우저에서 대시보드가 열리고 분석 결과를 확인할 수 있습니다.

### 사용된 기술 스택
- 언어: Python
- 프레임워크: Streamlit
- 데이터 처리: Pandas
- 시각화: Plotly (Express, Graph Objects)
